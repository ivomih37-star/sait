# Карточки-картинки для постов · Ракия Клуб · Москва

Готовые брендовые плашки **1080×1080** (квадрат для ленты/постов). Это
оформительская инфографика в стиле клуба — не фотографии.

> ℹ️ Фотореалистичные изображения генерируются во внешнем сервисе по промптам
> из [`../POST_IMAGES.md`](../POST_IMAGES.md). Эти PNG — быстрый брендовый
> вариант «здесь и сейчас».

## Файлы
- `card-grozdova.png` · `card-slivova.png` · `card-kaysieva.png`
- `card-dyuleva.png` · `card-vishneva.png` · `card-muskatova.png`
- `card-anons.png` — анонс первой дегустации

## Пересобрать / отредактировать
Данные карточек — в `generate.py` (название, крепость, сырьё, вкус, подача,
цвет акцента). После правок:

```bash
bash build.sh        # сгенерит HTML и отрендерит PNG через Chromium
```

Переменная `CHROME` задаёт путь к браузеру, если он не в PATH.

⚠️ 18+. Употребление алкоголя вредит вашему здоровью.
