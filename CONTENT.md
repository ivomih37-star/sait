# 📚 Материалы клуба · Ракия Клуб Москва

Сводная навигация по всем материалам проекта «Клуб любителей болгарской ракии».
Здесь — сайт, контент для соцсетей, орг-документы и печатные бланки.

> 🔁 Общие плейсхолдеры к замене во всех материалах: `@raki_club_msk` (канал),
> `@admin_username` (админ), `hello@raki-club.ru` (почта). Везде помним **18+**.

## 🌐 Сайт

| Файл | Что внутри |
|---|---|
| [`README.md`](README.md) | Запуск, сборка и деплой сайта (Next.js → GitHub Pages) |

**Запуск:** `npm install && npm run dev` → http://localhost:3000
**Сборка:** `npm run build` → каталог `./out`. Пуш в `main` деплоит на Pages.

## ✅ С чего начать (быстрый старт)

1. Завести канал и чат в Telegram, проставить плейсхолдеры во всех файлах.
2. Закрепить правила из [`CHAT_RULES.md`](CHAT_RULES.md) и настроить
   приветствие из [`WELCOME_MESSAGE.md`](WELCOME_MESSAGE.md).
3. Запустить прогрев по [`MARKETING.md`](MARKETING.md) (картинки — из
   [`POST_IMAGES.md`](POST_IMAGES.md)).
4. За 2 недели до встречи — закупка по [`TASTING_BUDGET.md`](TASTING_BUDGET.md),
   распечатать карты из [`tasting-card/`](tasting-card/).
5. Опубликовать сайт и держать актуальным расписание.
