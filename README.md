# Клуб любителей болгарской ракии · Москва

Сайт сообщества ценителей болгарской ракии в Москве: дегустации, встречи,
культура и традиции.

📚 Все материалы проекта (контент, правила, смета, бланки) — в [`CONTENT.md`](CONTENT.md).

Сделан на [Next.js](https://nextjs.org/) со статическим экспортом и
автоматически деплоится на GitHub Pages (см. `.github/workflows/nextjs.yml`).

## Локальный запуск

```bash
npm install
npm run dev      # http://localhost:3000
```

## Сборка статики

```bash
npm run build    # результат в каталоге ./out
```

## Деплой

Пуш в ветку `main` запускает GitHub Actions, который собирает сайт и
публикует его на GitHub Pages.

## Что поправить под себя

- Контакты (Telegram, почта) — в `app/page.js`.
- Тексты разделов, расписание встреч и виды ракии — там же.
- Цвета и оформление — в `app/globals.css`.
