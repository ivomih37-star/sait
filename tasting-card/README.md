# Карта дегустации · Ракия Клуб Москва

Печатный бланк для встреч клуба (формат A4, на одного участника).

- **`tasting-card.pdf`** — готовый файл, печатать по числу гостей.
- **`tasting-card.html`** — исходник. Правьте тексты/число образцов здесь.

## Что на карте
- Поля: имя, дата, тема встречи.
- 5 образцов: вид/название/регион, аромат · вкус · крепость · послевкусие,
  оценка ★ /5.
- Блок «Фаворит вечера» и памятка «как оценивать».

## Изменить число образцов
В `tasting-card.html` найдите `for (var i = 1; i <= 5; i++)` и поменяйте `5`.

## Пересобрать PDF
```bash
chrome --headless --no-sandbox --no-pdf-header-footer \
  --print-to-pdf=tasting-card.pdf tasting-card.html
```
(или откройте HTML в браузере → «Печать» → «Сохранить как PDF», поля «нет»).

⚠️ 18+. Употребление алкоголя вредит вашему здоровью.
